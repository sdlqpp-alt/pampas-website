# Pampas Engineering Website

## Quick launch
1. Upload this folder to GitHub.
2. Import the repo into Vercel.
3. Click Deploy.
4. Bind the domain `pampasgrp.com` in Vercel.
5. Replace temporary WhatsApp, LinkedIn and any placeholder project materials with final official content.

## Local run
```bash
npm install
npm run dev
```

## Current positioning
This site is built around one idea:

**Clients are not just looking for products. They are looking for a partner who can get the project done.**

The site therefore emphasizes:
- Building projects
- Solar projects
- Industrial projects
- One-stop supply across design, materials, production and delivery
